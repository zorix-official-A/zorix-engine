# Zorix Engine Feature Map

Zorix Engine maps common 3D editor concepts into a browser-first workflow:

| Editor area | Zorix feature |
| --- | --- |
| Hierarchy | Scene object list, selection, enable state |
| Scene view | WebGL viewport, orbit camera, transform gizmo |
| Game view | Runtime preview using the same scene |
| Inspector | Transform, material, physics, font and ZorixScript editing |
| Project | Prefabs, importers, materials, scene and build entries |
| Console | Timestamped runtime/editor log |
| Animator | Script-driven clips and behaviours |
| Physics | Gravity, collider, mass, bounciness |
| Lighting | Directional, hemisphere, point and spot lights |
| Environment | Procedural sky, fog, imported sky images |
| Assets | GLB/GLTF import, sky image import, starter content pack |
| Build | Static WebGL/browser package |

This project is a Zorix-branded browser engine inspired by common 3D editor workflows.
