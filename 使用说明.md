# Zorix Engine 使用说明

## 1. 覆盖安装

把代码覆盖包解压到临时目录，再复制到你已有的 `zorix-engine/zorix-engine` 目录。

```bash
cd ~/storage/downloads
rm -rf zorix-zscript-temp
mkdir zorix-zscript-temp
tar -xzf zorix-engine-zscript-full-code.tar.gz -C zorix-zscript-temp
cp -r zorix-zscript-temp/zorix-engine/* ~/storage/downloads/zorix-engine/zorix-engine/
```

你的最终目录应该包含：

```text
index.html
src/
vendor/
docs/
assets/
README.md
package.json
```

## 2. 启动

```bash
cd ~/storage/downloads/zorix-engine/zorix-engine
python3 -m http.server 8080
```

浏览器打开：

```text
http://localhost:8080
```

## 3. 创建 3D 物体

顶部 `Create...` 下拉菜单可以创建：

```text
Cube, Sphere, Capsule, Cylinder, Cone, Torus, Terrain, Water,
Tree, Vehicle, House, Tower, Bridge, Coin, Enemy, Platform, Portal,
Particles, Point Light, Spot Light, Camera Marker
```

底部 `Project` 面板也可以点击 Prefab 创建物体。

## 3.1 创建项目

第一次打开会出现 `Create Zorix Project`：

- `3D Game`：默认 3D 场景
- `2D Game`：更适合平台跳跃、2D 物体、正交视角
- `UI App / PWA`：适合做按钮、文本、面板和可安装网页 App

如果不想创建新项目，点 `Open Current Scene`。

## 4. 使用资源包

如果你的目录里有：

```text
assets/zorix-content-pack/
```

引擎会自动检测它。底部 `Project` 面板会出现：

```text
Use Pack Sky
Terrain Albedo
Terrain Normal
Water Normal
Crate Material
Metal Material
Load Starter Scene
```

使用方法：

1. 先在左侧 Hierarchy 或中间场景里选中物体。
2. 点底部 Project 里的材质按钮。
3. 贴图会应用到当前选中物体。

示例：

- 选中 `Sculpted Terrain`，点 `Terrain Albedo`
- 选中 `Water Plane`，点 `Water Normal`
- 选中任意 Cube/Crate，点 `Crate Material`
- 点 `Use Pack Sky` 会把天空换成资源包天空图

## 5. Inspector 面板

右侧 Inspector 可以编辑：

- `Name` 名称
- `Transform` 位置、旋转、缩放
- `Material Color` 材质颜色
- `Roughness` 粗糙度
- `Metallic` 金属度
- `Rigidbody / Collider` 重力、碰撞、质量、弹性
- `ZorixScript` 脚本

## 6. ZorixScript 基础

选择物体后，在 Inspector 里找到 `Script Source`。

脚本支持：

```zorixscript
start:
  log hello

update:
  spin y 1.2
```

`start:` 只运行一次。  
`update:` 每帧运行。

## 7. 树左右移动

选中树，把 Script Source 改成：

```zorixscript
update:
  sway x 1.3 0.8
  spin z 0.12
```

含义：

- `sway x 1.3 0.8`：沿 X 轴左右移动
- `spin z 0.12`：轻微左右摇摆

## 8. 玩家控制

选中 Player，把 Script Source 改成：

```zorixscript
start:
  gravity on
  bounce 0.1

update:
  ifkey w move 0 0 -4
  ifkey s move 0 0 4
  ifkey a move -4 0 0
  ifkey d move 4 0 0
  ifkey space velocity 0 5 0
  camera follow 4 3 6
  camera look
```

按键：

```text
W/S/A/D 移动
Space 跳跃
```

## 9.1 UI Designer

底部切换到 `UI Designer`，可以创建：

```text
Add Button
Add Text
Add Panel
```

UI 会显示在 3D 视图上方，适合做：

- 开始按钮
- 菜单
- 血条/提示文字
- 手机游戏触摸 UI 原型

### 点按钮隐藏树

1. 底部切到 `UI Designer`
2. 点 `Add Button`
3. 点场景里的新按钮
4. 右侧 Inspector 设置：

```text
Action: hide
Target Object: Zorix Tree
```

5. 点 `Play`
6. 在游戏画面里点这个按钮，`Zorix Tree` 就会消失

其他动作：

```text
hide      隐藏目标
show      显示目标
toggle    切换显示/隐藏
spawn     生成物体
runScript 运行一行 ZorixScript
```

## 9.2 隐藏/显示物体

选中任意物体，右侧 Inspector 有：

```text
Hide
Show
```

点击 `Hide`，物体会隐藏。点击 `Show`，物体会恢复。

## 9.3 导出 PWA

底部切换到 `Build` 或 `UI Designer`，点击：

```text
Export PWA Files
```

会下载：

```text
manifest.webmanifest
sw.js
```

再点：

```text
Export Standalone HTML
```

会下载：

```text
pwa-index.html
```

这些文件可以放到 Zorix Engine 项目根目录，用来做可安装网页 App 原型。

## 10. 常见问题

### 页面没有样式，像纯 HTML

说明缺少：

```text
src/styles.css
src/main.js
vendor/
```

重新覆盖代码包。

### 资源包按钮没出现

确认目录存在：

```text
assets/zorix-content-pack/manifest.json
```

### GLB 导入失败

优先使用 `.glb`，文件不要太大。手机浏览器内存有限。

### 手机屏幕太窄

建议横屏使用，Inspector 和 Project 面板会更容易操作。
